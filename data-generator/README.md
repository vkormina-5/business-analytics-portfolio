# Synthetic Data Generator

This script generates a synthetic retail dataset for BI dashboards.

## Files Generated
- `products.csv` – 200 products with categories, brands, and pricing
- `stores.csv` – 50 stores across major cities
- `sales.csv` – ~100,000 transactions for 2023–2024
